About

Hi, my name is Kyla Qi. I'm currently attending Georgia Tech in pursuit of a B.S. in Computer Science. I'm interested in digital art, web design, and robotics.

My skills include a working knowledge of IntelliJ, Github, Inkscape, and EagleCAD, as well as a fluency in Java and C++ and a limited knowledge of HTML, CSS, and Python.

When not in class I'm at the Robojackets' shop as a part of their outreach subteam, or working on my own projects, most of which are currently learning driven and web dev related. Long term I hope to improve people's lives through combining data with machine learning solutions.
